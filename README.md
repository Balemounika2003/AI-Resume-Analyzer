# Resume Analyzer

## Overview

Resume Analyzer is a web application developed using React, Flask, and MySQL that helps users evaluate and improve their resumes. The system analyzes uploaded resumes, extracts skills, calculates ATS scores, identifies missing keywords, and provides recommendations to improve job readiness.

---

## Features

### User Features

* Upload Resume (PDF)
* ATS Score Calculation
* Skill Extraction
* Keyword Analysis
* Missing Skills Detection
* Resume Improvement Suggestions
* Job Role Prediction
* Dashboard View

## Workflow

1. User uploads a resume.
2. Resume text is extracted.
3. Skills and keywords are identified.
4. ATS score is calculated.
5. Missing skills are detected.
6. Job role is predicted.
7. Results are displayed on the dashboard.
